Case Study: Currency Converter App

Project Overview

Client: Personal Project

Duration: 4 Weeks

Role: UI/UX Designer

Tools: Figma, Vega-Lite

Background and Context

In an increasingly globalized world, the need to quickly and accurately convert currencies is more important than ever. Whether for business, travel, or online shopping, having access to up-to-date exchange rates is crucial. While there are numerous currency conversion tools available, many lack intuitive design and visual elements that help users understand exchange rate trends.

This project aims to create a user-friendly currency converter app with a sleek design and a graph feature to visualize currency exchange trends, providing users with both immediate conversion results and insights into historical data.

Objective

The primary goal of this project was to design a currency converter app that not only performs quick conversions but also includes a graph to display historical currency exchange rates. The app should be simple to use, visually appealing, and informative, offering a superior user experience compared to existing apps.

Research and Planning

  • User Interviews and Surveys: Gathered insights on user experiences with current currency conversion apps.
  • Competitive Analysis: Analyzed popular apps to identify common features, design patterns, and areas for improvement.
  • Persona Development: Created user personas to guide the design process based on research findings.

Design Process

1. Ideation and Sketching

Based on the insights gathered, I sketched out several design concepts focusing on simplicity and functionality. The main features identified were:

  • Currency Conversion Tool
  • Exchange Rate Graph

2. Wireframing and Prototyping

Using Figma, I developed wireframes to lay out the appâ€™s structure, including a simple home screen with an input field for selecting currencies and entering amounts, and a graph feature for displaying historical rates.

3. Adding the Graph Feature

To enhance the user experience, I integrated a graph feature using Vega-Lite, which displays a line graph of exchange rates over a customizable period.

4. Visual Design

The visual design focused on a clean, modern look with a minimalistic color palette to ensure readability and reduce cognitive load.

Outcome and Impact

The final design of the currency converter app was well-received by test users who appreciated the clean interface and the ability to view historical trends directly within the app.

Lessons Learned and Next Steps

Lessons Learned:

  • User Feedback is Key
  • Balancing Simplicity with Functionality

Next Steps:

  • Expand Features
  • Testing and Refinement
